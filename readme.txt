Add a custom scrollbar to your wordpress site.

== Description ==

This plugin allows you to set a Nice Custom ScrollBar which you can control from its Option Page.

== Features ==

*You can change Scrollbar speed.
*You can change Scrollbar Color, Unlimited color option.
*You can change Scrollbar width.
*You can change Scrollbar radius.
*You can change scrollbar border color.
*Control Scrollbar autohide enable or disable.
*Compatible with almost all Major Browser.
*Responsive.

== Installation ==

1. Upload `plugin-directory` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to Myscrollbar option and set the parameter.
